## Contributing

This site is powered by the [Jekyll](https://jekyllrb.com/) gem! You can see [their docs](https://jekyllrb.com/docs/) for more information.

You can view our contributing instructions at [https://pswifi.github.io/contributing/](https://pswifi.github.io/contributing/)

Broadly speaking, we welcome contributions that are:
- Permanent (Not something with frequent changes like a monthly schedule)
- Informative (Users will be looking at this site to find information about how the room works)
- Futureproof (Would only need minor tweaks to keep updated in future)
- Typo and/or namecolor css fixes
- Accessibility improvements


**Important note!**
The live preview will be generated in the `_site` folder. Any changes you make to files in this folder *will not be saved*!
