Cloning Information

Wondering what cloners do, or hoping to become an approved cloner? This page contains information about cloning. The PS! Wi-Fi room permits the use of cloning for legitimate Pokémon. Cloning is the act of copy/pasting already existing Pokémon data in order to have multiple copies. If done correctly the Pokémon data will not be altered, this remaining legitimate by our room standards.

We facilitate cloning in all main series Pokémon games that are currently supported by official Nintendo Online services. This includes all 3DS and Switch titles.

Cloning methods that alters Pokémon data in any way is strictly banned and will render the Pokémon as not legitimate.

Cloning Requirements:

Wi-Fi approved cloners must:
  1. Have a good general knowledge of room and proof rules.
  2. Use a cloning method that allows for quick/efficient hack checks.
  3. Be able to identify a hacked or illegal Pokémon to a good extent
  4. Be an active and positive presence in the room for an extended time period
Room/Proof rule knowledge and hack checking competency will be tested before you are approved for cloning! This is an open-book test so you will be able to refer to any information you need while the test is ongoing.

Important Information:

  • Poor behaviour will result in removal from our approved cloning team.
  • Scamming or trading hacked Pokémon in a position of trust will result in immediate removal and room punishment.
  • If you need a second opinion on a hack check, approach a member of /roomstaff. (% or higher)
  • If you get a suspicious cloning request, clone the Pokémon, return it, then approach a staff member with the Pokémon and logs of the conversation.

Approved Cloning Methods:

This section of the page will cover all the cloning methods we allow, and which ones are immediately disallowed.

Switch:
  • PKHeX - We allow and recommend cloning using PKHeX as the tool is widely known and accepted for making accurate clones of Pokémon. It is extremely important to not misuse PKHeX as you will be held responsible for any misuse.
  • Livehex Livehex is a plugin used with PKHeX which allows for cloning remotely while the game is running. This can also be used for cloning/hack checking Pokemon mid-trade.
3DS:
  • PKHeX - We allow and recommend cloning using PKHeX as the tool is widely known and accepted for making accurate clones of Pokémon. It is extremely important to not misuse PKHeX as you will be held responsible for any misuse.
  • Livehex Livehex is a plugin used with PKHeX which allows for cloning remotely while the game is running. This can also be used for cloning/hack checking Pokemon mid-trade.
  • PKSM - PKSM is a tool used for cloning on-the-fly on a 3DS console, this allows for basic hack checking and is an approvable method by our room standard.
Unapprovable Cloning Methods:
  • Action Replay
  • Powersaves
  • God Egg
  • Other alternative cheating engines/tools/glitches.
  • Someone elses tool, a bot for example.
